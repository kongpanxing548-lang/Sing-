# Sing+ 音乐平台 - 项目规格文档

## 项目概述

- **项目名称**: Sing+ (音乐平台)
- **版本**: V0.1
- **类型**: 音乐类综合网站（史诗音乐 + 音乐资讯）
- **参考**: Audiomachine（视觉风格）+ Sonixa（结构布局）

## V0.1 规格

### 1. 页面结构

```
首页
├── Hero Section（全屏视觉冲击）
├── 导航栏（固定顶部 + 页签分类）
├── 最新发行（Latest Releases）
├── 音乐分类（Categories/Tags）
├── 精选专辑（Featured Albums）
├── 演出活动（Upcoming Events）
├── 关于我们（About）
└── 页脚（Footer + Social Links）
```

### 2. 视觉规格

**色彩方案**（史诗音乐风格）
- 主背景: `#0a0a0f` (深黑)
- 主色调: `#ff3d00` (炽热橙红 - 视觉冲击)
- 强调色: `#00e5ff` (电光青)
- 文字色: `#ffffff` (白) / `#b0b0b0` (灰)
- 渐变: `linear-gradient(135deg, #ff3d00 0%, #ff6f00 100%)`

**字体**
- 标题: 'Orbitron', sans-serif (科技感)
- 正文: 'Rajdhani', sans-serif (现代)
- 字重: 700 (标题), 400/500 (正文)

**视觉效果**
- 背景: 深色渐变 + 动态粒子/光效
- 卡片: 玻璃拟态效果 (backdrop-filter)
- 动画: 滚动淡入、悬停缩放
- 边框: 1px 发光边框

### 3. 导航栏规格

左侧: Logo + 品牌名
右侧菜单项:
- 首页 (Home)
- 音乐 (Music) - 下拉分类
- 专辑 (Albums)
- 演出 (Events)
- 关于 (About)
- 搜索图标
- 登录/注册按钮

### 4. 功能规格

**Hero Section**
- 全屏背景（动态渐变/视频）
- 主标题 + 副标题
- CTA 按钮（立即体验/了解更多）

**最新发行**
- 卡片式布局（3-4列）
- 封面图 + 标题 + 艺术家
- 悬停播放效果

**音乐分类**
- 标签云形式
- 分类: 史诗/电子/嘻哈/古典/氛围/流行

**演出活动**
- 时间线布局
- 地点 + 日期 + 购票按钮

### 5. 响应式断点

- 桌面: > 1200px
- 平板: 768px - 1200px
- 手机: < 768px

## 验收标准

1. ✅ 页面加载无报错
2. ✅ 导航栏固定且可点击
3. ✅ Hero 区域视觉冲击力强
4. ✅ 音乐分类标签可交互
5. ✅ 响应式布局正常
6. ✅ 动画效果流畅
