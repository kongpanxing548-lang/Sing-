# NovaSynth 开发日志

## 项目信息
- **名称**: NovaSynth - Web 音频合成器
- **目标**: 比 CrystalPrism 更强大的 Web 合成器
- **状态**: Phase 1 完成 ✅

## 开发阶段

### Phase 1: 基础版 ✅
- [x] 项目初始化 + 技术架构
- [x] 基础合成引擎（Tone.js）
- [x] 音色库（50+ 预设，7个分类）
- [x] 基础 UI 框架
- [x] 3D 粒子可视化（Three.js）
- [x] 虚拟键盘（鼠标+计算机键盘）
- [x] 控制面板（音量、BPM、录音）

### Phase 2: 效果器
- [ ] 效果器链架构
- [ ] EQ / Compressor / Delay / Reverb

### Phase 3: 可视化
- [ ] 2D 示波器
- [ ] 3D 粒子系统增强

### Phase 4: 高级功能
- [ ] 琶音器 + 音序器
- [ ] MIDI 支持
- [ ] 录音导出

### Phase 5: 生态
- [ ] 预设分享
- [ ] 移动端适配

## 技术栈
- React 18 + TypeScript + Vite
- Tone.js (Web Audio)
- Three.js (3D 可视化)
- Zustand (状态管理)
